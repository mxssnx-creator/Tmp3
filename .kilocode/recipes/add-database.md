# Add Database Recipe

## Overview
When users need data persistence for users, posts, or other entities, this recipe provides implementation guidance.

## Implementation Steps
1. Define database schema and models
2. Set up connection pooling
3. Implement CRUD operations
4. Add migration system
5. Configure backup strategy
6. Set up monitoring and logging

## Related Files
- Database configuration: `.kilocode/config/database.js`
- Migration files: `.kilocode/migrations/`
- Models: `.kilocode/models/`
